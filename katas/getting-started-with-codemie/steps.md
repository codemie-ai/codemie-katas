## üéØ Challenge 1: Master the Sidebar Navigation

**Goal:** Master the left sidebar navigation and discover the key features available in each platform section

### Instructions

The menu on the left sidebar is your command center‚Äîplacing all essential tools right at your fingertips!

1. **Open CodeMie** in your browser and notice the left sidebar
2. **Locate the "Expand Menu" button** under your profile icon
3. **Click "Expand Menu"** to see all available tools with their names

 **Explore the structured layout:** Chats, Assistants, Workflows, Integrations, Data Sources, pre-built Assistants, Help Center, Profile
![Left sidebar](https://raw.githubusercontent.com/codemie-ai/codemie-katas/refs/heads/main/katas/getting-started-with-codemie/images/left_sidebar.png)

   **Now let's discover what each section contains!**

4. **Click on "Chats"**:
At the top, you'll find the **"+ New Chat"** button‚Äîclicking it automatically creates a new chat. If you want to communicate with a different assistant, simply type `@AssistantName` in the chat window followed by your request.

**ASSISTANTS**: Shows your recently used assistants. Click the 3-dots menu next to any assistant to create a new chat with them, view assistant details, or remove them from the list. Click **"Explore Assistants"** to navigate to the full Assistants marketplace.

**WORKFLOWS**: Displays your workflow conversations. Click **"Explore Workflows"** to explore Workflow page.

By expanding **Chats list** you'll be able to view the list of chats.
When you expand the **Folders list**, you'll see folders named after the assistants used in chats. For example, if you interact with the AI/Run FAQ Assistant, a folder AI/Run FAQ appears with the relevant chats inside.

5. **Click on "Assistants"**:
   
   The Assistants menu is your gateway to discovering, creating and managing AI assistants. It contains three main tabs:
   
   - **Project Assistants**: Displays all assistants available in your assigned project. You can filter assistants using the search option, by selecting specific projects (if you're assigned to multiple), by creator, or by shared/not shared status. Use the "Created by me" checkbox to see only your own assistants.
   
   - **Marketplace**: A collection of pre-built, ready-to-use solutions contributed by the CodeMie community. Use the search function to quickly find assistants tailored to your needs, use the search bar, try the categories filter, filter by creator, or use the "Created by Me" checkbox to find your published assistants.
   
   - **Templates**: Provides a list of all Assistant templates available to you as starting points for creating custom assistants.

6. **Click on "Workflows"**:
   
   The Workflows menu (Explore Workflows page) is your automation hub with three main tabs:
   
   - **My Workflows**: Shows the list of workflows you have created.
   
   - **All Workflows**: Displays all workflows created by you, plus workflows created and shared by other teammates within your available projects.
   
   - **Templates**: Provides a list of all predefined workflow templates available on the platform as starting points.
   
   In the top-right corner, you'll find the **"+ Create Workflow"** button that opens the Workflow constructor, allowing you to build custom multi-step automations. Browse through each tab to see available workflows and templates.

7. **Click on "Integrations"**:
    
    The Integrations page helps you manage connections to external tools and services. At the top right, you'll see:
    
    - **Integration Type toggle**: Switch between "User" and "Project" integrations (available for users with admin rights)
    - **"+ Create"** button: Add new integrations to connect external tools
    
    On the left side, you'll find **FILTERS** panel:
    - **Search**: Quickly find specific integrations by alias
    - **PROJECT**: Filter by project name
    - **TYPE**: Filter by integration type (e.g., Jira)
    - **GLOBAL**: Filter by scope (All, Global, Non Global)
    
    The main table displays integrations created by you or your project team.

8.  **Click on "Data Sources"**:
    
    The Data Sources page helps you connect and manage your data sources in one place. At the top right, you'll find the **"+ Create Datasource"** button to add new data sources.
    
    On the left side, there's a **FILTERS** panel to organize your data sources:
    - **Search**: Quickly find specific data sources by name
    - **TYPE**: Filter by data source type with checkboxes
    - **PROJECT**: Filter by project dropdown
    - **CREATED BY**: Filter with "Created by Me" checkbox or dropdown by creator
    - **STATUS**: Filter by data source status dropdown
    
    The main table displays data sources created by you or your project team
    
9. **Try the "Hide Menu" option** to collapse the sidebar when you need more screen space
**üí° Pro Tip:** The left sidebar is always accessible from any page‚Äîuse it as your quick navigation hub to jump between Chats, Assistants, Workflows, and resources without losing your context or breaking your workflow!

**‚úÖ Success Criteria:**
- [ ] Located the left sidebar and expanded the menu to see all sections
- [ ] Explored the Chats section and understood its organization
- [ ] Navigated to the Assistants menu and reviewed all three tabs (Project Assistants, Marketplace, Templates)
- [ ] Visited the Workflows section and reviewed all three tabs (My Workflows, All Workflows, Templates)
- [ ] Explored Integrations page with available integration types
- [ ] Explored Data Sources page with available data source types
- [ ] Understand the purpose and key features of each main platform section

**üèÜ Bonus Challenge:**
- Search for at least 3 different assistants from the Marketplace tab
- Review one Assistant template to understand how it can jumpstart your custom assistant creation
- Explore one workflow template and understand its use case
- Click the "+ Create Datasource" button to explore the data source creation page and see what types you can connect
- Click the "+ Create" button to explore the integration creation process and see what types of integrations you can set up

## üéØ Challenge 5: Master Chat Organization and Configuration

**Goal:** Learn to organize conversations using the folder system and configure assistants without leaving the chat page

### Instructions

The Chats menu in the left sidebar includes intelligent chat organization that keeps your conversations structured and easy to find. Plus, you can configure assistants on-the-fly without interrupting your workflow!

#### Part 1: Organize Chats with Smart Folders

1. **Navigate to the left sidebar** where all your chats are stored
![chats](https://raw.githubusercontent.com/codemie-ai/codemie-katas/refs/heads/main/katas/getting-started-with-codemie/images/chats.png)
2. **Create a chat** with a previously discovered Pre-built assistants by clicking on one of them
3. **Observe the automatic folder creation**:
   - When you expand the **FOLDERS** list, you'll see folders named after the assistants used in chats
   - For example, if you interact with the AI/Run FAQ Assistant, a folder "AI/Run FAQ" appears with the relevant chats inside
   - All chats with that assistant are automatically grouped inside

4. **Rename a chat**:
   - Click the three-dots menu on a chat name and select **Rename** and give it a descriptive name.

5. **Rename a folder**:
   - Use the three-dots menu to edit an assistant folder name to match your needs

6. **Create a new custom folder**:
   - Use the **Create Folder** icon to create a "New Folder"
   - Name it according to your project or workflow

7. **Move chats between folders** to test the organization system and keep related conversations together

#### Part 2: Configure Assistants Without Leaving Chat

8. **While in a chat**, locate the **Configuration** button in the upper right corner of the chat window

9. **Click the Configuration button**:
   - A sidebar opens on the right with all the settings you need
   - Notice how you can adjust settings without losing your chat context
![configuration](https://raw.githubusercontent.com/codemie-ai/codemie-katas/refs/heads/main/katas/getting-started-with-codemie/images/—Åonfiguration.png)

10. **Explore the General Section**:
    - Select the LLM model that best fits your needs

11. **Review Connected Assistance settings**, if you are the creator of the assistant or have project admin rights you can:
    - Adjust general assistant details
    - Modify the system instructions
    - Add data sources and integrations directly from the sidebar
